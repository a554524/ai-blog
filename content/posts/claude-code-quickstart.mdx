## 什么是 Claude Code

Claude Code 是 Anthropic 官方的终端 AI 编码助手,能读写你的文件、跑命令、接 MCP 服务器。相比网页聊天,它能直接操作代码库。

## 安装

```bash
npm install -g @anthropic-ai/claude-code
```

或者用 Homebrew:

```bash
brew install claude-code
```

## 第一次使用

进入项目目录,运行:

```bash
cd your-project
claude
```

首次会引导你登录。之后就可以自然语言下指令:

> 帮我把 src/utils 下所有的 var 改成 const,然后跑测试

## 几个常用技巧

- **`/clear`** 清空上下文,开始新任务
- **`@path/to/file`** 直接引用文件
- **`!command`** 执行 shell 命令
- **`/plan`** 进入只读规划模式
